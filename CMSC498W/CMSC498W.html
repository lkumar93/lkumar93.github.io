About The Project

The main idea behind this project is to demonstrate a human-robot interaction scenario in a modern assembly line where a human can teach a robot to assemble. The human end-user manipulates scale models (CAD models) of parts to be assembled,using the Leap Motion controller as an input device. The robot manipulator simulator "mirrors" these actions on full-scale parts.The entire simulation takes place in a virtual environment, rendered using the Unity Game Engine on an Oculus Rift DK2.
